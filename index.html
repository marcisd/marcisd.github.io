All Aboard: Alphabet and early reading app

⚓ All Aboard

Unity EdTech Team Lead at All Aboard Learning

All Aboard is a complete systematic synthetic phonics (SSP) teaching tool, designed by teachers to make phonics sessions easier to deliver and progress more consistent across every class.

Key Features

Reading-Based Games. A plethora of fun minigames and activities that is structured around letter recognition, phoneme sound recognition, and reading and segmenting words.

Regionalized Curricula. Designed to work in harmony with different dialects and with different school system requirements around the world.

Decodable Ebook Library. A complete set of illustrated short decodable ebooks that integrate with the curriculum.

Learner Management System. A dashboard for parent and teacher facilitators to manage and monitor the learners and groups in their accounts.

Crisalix 4D Augmented Reality Live VR

🥽 Crisalix

3D Software Developer at Crisalix

Crisalix uses artificial intelligence, computer vision, and augmented reality in the creation of 3D reconstructions of human bodies and the simulation of aesthetic procedures on top.

Key Contributions

Scars Simulation. This feature enables users to create freehand drawings of scars on the patient's simulation and provides a projection of how the scar will heal over time.

Mirroring Tool. This feature is utilized in reconstructive procedures, using one half of the body as a reference to restore or correct deformities on the other half. It is also used to simulate asymmetry correction procedures.

Face Implants. This feature enables the user to select from a catalog of forhead, nose, chin, and cheek implants and apply it to the patient's face simulation.

🕶 Tech Demo

💎 SameGame Demo

I chose the game SameGame for this demo because of its simple game design and simple architecture. It was also an opportunity for me to revisit an old project and explore how I can expand my knowledge of artificial intelligence and how it is used generate adaptive, responsive, and intelligent game simulations.

The Monte Carlo Tree Search algorithm is used to explore an approximation of the best possible moves. The Greedy Best-First Search heuristics is used in expanding towards most promising moves.

🀄 Custom 2D Orthogonal Tile Map in Unity

Inspired by a lot of 2D indie games as well as the classic ones, I did an exploration on the implementation tile-based graphics in Unity. This project also explores the use of auto-tiling or the technique of automating the drawing of tile graphics based on its surroundings.

This was made prior to the addition of Unity's Tile Map package that was introduced in Unity version 2017.2.
